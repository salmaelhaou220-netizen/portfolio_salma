(fichiers séparés : index.html, style.css, app.js).

Je suis Salma, professeure stagiaire en Informatique au CRMEF Rabat (2026),
en stage au Lycée Hommane El Fetouaki à Rabat. Master 2 Cloud Computing en cours.

=== LOGIQUE FICHIERS (IMPORTANTE) ===
Ce portfolio est PUBLIC. Les visiteurs (jury, formateurs) peuvent UNIQUEMENT
visualiser et télécharger les fichiers. Il n'y a AUCUN bouton d'upload,
AUCUN drag & drop. Tous les fichiers sont hébergés directement dans le projet
dans un dossier /docs/ organisé en sous-dossiers par catégorie.

Structure des fichiers dans le projet :
docs/fiches/     → fiches de préparation (PDF, DOCX)
docs/grilles/    → grilles d'observation (JPG, PDF)
docs/supports/   → supports de cours (PDF, PPTX)
docs/pff/        → projet de fin de formation (PDF)
docs/rapport_stage/        → projet de fin de formation (PDF)
docs/productions/→ autres productions (PDF)

Chaque document dans le code est déclaré comme un objet avec :
{ name, description, date, type, path: 'docs/fiches/nom_fichier.pdf' }

Le path est un chemin relatif vers le vrai fichier dans le projet.

=== FONCTIONNALITÉS DOCUMENTS ===
- Bouton 👁 VISUALISER : ouvre une modale avec iframe pour PDF,
  ou balise <img> pour les images. Plein écran, fermeture par Escape ou clic overlay.
- Bouton ⬇ TÉLÉCHARGER : crée un lien <a href="docs/..." download> et déclenche le clic.
- PAS de suppression, PAS d'upload, PAS de drag & drop.
- Les fichiers sont en lecture seule pour les visiteurs.
